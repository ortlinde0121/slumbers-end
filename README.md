### 📖 故事

哥哥满身是血地推开家门，反锁，告诉我丧尸末日爆发了。

三年里，我深信不疑。我看着他冒着生命危险去找物资，我帮他包扎伤口，为他掉眼泪，全心全意地依赖他。

直到我发现血是假的。红酒。糖浆。甜的。

伤是假的，血是假的——那末日呢？

---

### 🕹️ 玩法

- **纯文字。** 阅读故事，做出选择，看着后果一步步展开。
- **4条主线 + 2个二级分支 = 6条阅读路径。**
- **4个结局。**
- 单条路线约 **10分钟**。
- 全路线+隐藏内容收集约 **1小时**。

---

### ✨ 系统

| 系统 | 说明 |
|------|------|
| 🧭 **路线图** | 每通关一条路线解锁对应节点。随时查看已读和未读。 |
| 🔁 **二周目** | 通关任意结局后重新开始，文本中会浮现一周目不曾出现的追加伏笔。 |
| 📦 **实物** | 每条路线解锁一件隐藏道具文本——收据、日记、标签……藏在别墅某处。 |
| 🧩 **深层记忆** | 收集全部6条路线后解锁：哥哥的六段碎忆、一封从未送出的信、和最后的觉醒。 |

### 🎭 人物

- **主角** — 你。无性别设定。
- **哥哥** — 擅长用委屈和眼泪操纵人心。一个发现谎话被拆穿后不暴怒、只会委屈巴巴的顶级病娇男绿茶。
